# Restaurant Allergen Audit Platform

## Summary
Restaurant Allergen Audit Platform is a full-stack workspace for managing restaurant allergen workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Restaurant Allergen Audit Platform addresses a real-world gap in safety compliance, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Restaurant Allergen Audit Platform audit dashboard
- Restaurant Allergen Audit Platform ingredient matrix
- Restaurant Allergen Audit Platform incident board

## Backend Features
- Restaurant Allergen Audit Platform audit APIs
- Restaurant Allergen Audit Platform incident workflows
- Restaurant Allergen Audit Platform corrective action logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
