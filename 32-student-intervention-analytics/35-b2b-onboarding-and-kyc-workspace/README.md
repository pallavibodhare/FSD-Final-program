# B2B Onboarding and KYC Workspace

## Summary
B2B Onboarding and KYC Workspace is a full-stack workspace for managing b2b onboarding workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
B2B Onboarding and KYC Workspace addresses a real-world gap in business onboarding, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- B2B Onboarding and KYC Workspace onboarding checklist
- B2B Onboarding and KYC Workspace review dashboard
- B2B Onboarding and KYC Workspace document timeline

## Backend Features
- B2B Onboarding and KYC Workspace applicant APIs
- B2B Onboarding and KYC Workspace verification workflows
- B2B Onboarding and KYC Workspace risk notes

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
