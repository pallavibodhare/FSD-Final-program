# Student Intervention Analytics

## Summary
Student Intervention Analytics is a full-stack workspace for managing student intervention workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Student Intervention Analytics addresses a real-world gap in student intervention, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Student Intervention Analytics risk dashboard
- Student Intervention Analytics cohort charts
- Student Intervention Analytics counselor timeline

## Backend Features
- Student Intervention Analytics student signal APIs
- Student Intervention Analytics intervention workflows
- Student Intervention Analytics risk summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
