# Grant Compliance Tracker

## Summary
Grant Compliance Tracker is a full-stack workspace for managing grant compliance workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Grant Compliance Tracker addresses a real-world gap in grant management, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Grant Compliance Tracker grant timeline
- Grant Compliance Tracker compliance board
- Grant Compliance Tracker evidence uploads

## Backend Features
- Grant Compliance Tracker grant APIs
- Grant Compliance Tracker milestone endpoints
- Grant Compliance Tracker reporting summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
