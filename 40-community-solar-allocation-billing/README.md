# Community Solar Allocation Billing

## Summary
Community Solar Allocation Billing is a full-stack workspace for managing community solar workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Community Solar Allocation Billing addresses a real-world gap in allocation billing, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Community Solar Allocation Billing member dashboard
- Community Solar Allocation Billing ledger views
- Community Solar Allocation Billing billing board

## Backend Features
- Community Solar Allocation Billing allocation APIs
- Community Solar Allocation Billing invoice endpoints
- Community Solar Allocation Billing settlement summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
