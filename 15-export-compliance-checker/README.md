# Export Compliance Checker

## Summary
Export Compliance Checker is a full-stack workspace for managing export compliance workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Export Compliance Checker addresses a real-world gap in shipment compliance, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Export Compliance Checker review dashboard
- Export Compliance Checker document checklist
- Export Compliance Checker flagged issues board

## Backend Features
- Export Compliance Checker shipment APIs
- Export Compliance Checker validation endpoints
- Export Compliance Checker compliance logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
