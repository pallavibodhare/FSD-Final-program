# Election Volunteer Logistics Console

## Summary
Election Volunteer Logistics Console is a full-stack workspace for managing election volunteer workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Election Volunteer Logistics Console addresses a real-world gap in city coordination, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Election Volunteer Logistics Console operations map
- Election Volunteer Logistics Console route board
- Election Volunteer Logistics Console issue dashboard

## Backend Features
- Election Volunteer Logistics Console city service APIs
- Election Volunteer Logistics Console routing endpoints
- Election Volunteer Logistics Console trend metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
