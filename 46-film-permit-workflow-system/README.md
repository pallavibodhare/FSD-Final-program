# Film Permit Workflow System

## Summary
Film Permit Workflow System is a full-stack workspace for managing film permit workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Film Permit Workflow System addresses a real-world gap in permit approvals, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Film Permit Workflow System application dashboard
- Film Permit Workflow System location review views
- Film Permit Workflow System approval timeline

## Backend Features
- Film Permit Workflow System permit APIs
- Film Permit Workflow System approval workflows
- Film Permit Workflow System requirement logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
