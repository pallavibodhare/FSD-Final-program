# EV Fleet Charge Optimizer

## Summary
EV Fleet Charge Optimizer is a full-stack workspace for managing ev fleet workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
EV Fleet Charge Optimizer addresses a real-world gap in fleet planning, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- EV Fleet Charge Optimizer vehicle dashboard
- EV Fleet Charge Optimizer planning board
- EV Fleet Charge Optimizer cost projections

## Backend Features
- EV Fleet Charge Optimizer fleet APIs
- EV Fleet Charge Optimizer schedule endpoints
- EV Fleet Charge Optimizer utilization metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
