# Supply Chain Provenance Ledger

## Summary
Supply Chain Provenance Ledger is a full-stack workspace for managing supply chain workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Supply Chain Provenance Ledger addresses a real-world gap in provenance tracking, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Supply Chain Provenance Ledger trace explorer
- Supply Chain Provenance Ledger supplier cards
- Supply Chain Provenance Ledger verification views

## Backend Features
- Supply Chain Provenance Ledger lot APIs
- Supply Chain Provenance Ledger event workflows
- Supply Chain Provenance Ledger certification summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
