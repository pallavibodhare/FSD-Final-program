# Refugee Case Coordination Platform

## Summary
Refugee Case Coordination Platform is a full-stack workspace for managing refugee case workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Refugee Case Coordination Platform addresses a real-world gap in case coordination, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Refugee Case Coordination Platform support dashboard
- Refugee Case Coordination Platform referral board
- Refugee Case Coordination Platform document timeline

## Backend Features
- Refugee Case Coordination Platform case APIs
- Refugee Case Coordination Platform referral workflows
- Refugee Case Coordination Platform follow-up metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
