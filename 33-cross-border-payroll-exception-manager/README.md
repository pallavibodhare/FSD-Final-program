# Cross-Border Payroll Exception Manager

## Summary
Cross-Border Payroll Exception Manager is a full-stack workspace for managing cross border workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Cross-Border Payroll Exception Manager addresses a real-world gap in payroll exception handling, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- CrossBorder Payroll Exception Manager country issue board
- CrossBorder Payroll Exception Manager impact cards
- CrossBorder Payroll Exception Manager review workflow views

## Backend Features
- CrossBorder Payroll Exception Manager payroll issue APIs
- CrossBorder Payroll Exception Manager rule metadata
- CrossBorder Payroll Exception Manager resolution logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
