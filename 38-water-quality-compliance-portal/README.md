# Water Quality Compliance Portal

## Summary
Water Quality Compliance Portal is a full-stack workspace for managing water quality workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Water Quality Compliance Portal addresses a real-world gap in quality compliance, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Water Quality Compliance Portal sampling dashboard
- Water Quality Compliance Portal trend views
- Water Quality Compliance Portal violation board

## Backend Features
- Water Quality Compliance Portal sample APIs
- Water Quality Compliance Portal threshold checks
- Water Quality Compliance Portal notice endpoints

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
