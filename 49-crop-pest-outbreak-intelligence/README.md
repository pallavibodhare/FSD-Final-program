# Crop Pest Outbreak Intelligence

## Summary
Crop Pest Outbreak Intelligence is a full-stack workspace for managing crop pest workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Crop Pest Outbreak Intelligence addresses a real-world gap in outbreak intelligence, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Crop Pest Outbreak Intelligence field dashboard
- Crop Pest Outbreak Intelligence alert map
- Crop Pest Outbreak Intelligence advisory panels

## Backend Features
- Crop Pest Outbreak Intelligence field APIs
- Crop Pest Outbreak Intelligence risk aggregation
- Crop Pest Outbreak Intelligence alert endpoints

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
