# Shift Swap With Skill Constraints

## Summary
Shift Swap With Skill Constraints is a full-stack workspace for managing shift swap workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Shift Swap With Skill Constraints addresses a real-world gap in staff scheduling, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Shift Swap With Skill Constraints schedule grid
- Shift Swap With Skill Constraints request board
- Shift Swap With Skill Constraints approval inbox

## Backend Features
- Shift Swap With Skill Constraints shift APIs
- Shift Swap With Skill Constraints validation rules
- Shift Swap With Skill Constraints staffing summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
