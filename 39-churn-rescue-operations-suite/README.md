# Churn Rescue Operations Suite

## Summary
Churn Rescue Operations Suite is a full-stack workspace for managing churn rescue workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Churn Rescue Operations Suite addresses a real-world gap in retention operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Churn Rescue Operations Suite account dashboard
- Churn Rescue Operations Suite rescue board
- Churn Rescue Operations Suite health timeline

## Backend Features
- Churn Rescue Operations Suite account APIs
- Churn Rescue Operations Suite risk endpoints
- Churn Rescue Operations Suite playbook workflows

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
