# Scholarship Disbursement Manager

## Summary
Scholarship Disbursement Manager is a full-stack workspace for managing scholarship disbursement workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Scholarship Disbursement Manager addresses a real-world gap in funding operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Scholarship Disbursement Manager award timeline
- Scholarship Disbursement Manager student status cards
- Scholarship Disbursement Manager approvals dashboard

## Backend Features
- Scholarship Disbursement Manager award APIs
- Scholarship Disbursement Manager eligibility checks
- Scholarship Disbursement Manager ledger summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
