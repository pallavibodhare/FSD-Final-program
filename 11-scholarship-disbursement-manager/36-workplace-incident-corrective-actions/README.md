# Workplace Incident Corrective Actions

## Summary
Workplace Incident Corrective Actions is a full-stack workspace for managing workplace incident workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Workplace Incident Corrective Actions addresses a real-world gap in incident prevention, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Workplace Incident Corrective Actions incident dashboard
- Workplace Incident Corrective Actions action board
- Workplace Incident Corrective Actions trend summaries

## Backend Features
- Workplace Incident Corrective Actions incident APIs
- Workplace Incident Corrective Actions corrective workflows
- Workplace Incident Corrective Actions closure metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
