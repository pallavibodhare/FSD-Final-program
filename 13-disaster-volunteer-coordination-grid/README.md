# Disaster Volunteer Coordination Grid

## Summary
Disaster Volunteer Coordination Grid is a full-stack workspace for managing disaster volunteer workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Disaster Volunteer Coordination Grid addresses a real-world gap in public operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Disaster Volunteer Coordination Grid command dashboard
- Disaster Volunteer Coordination Grid service board
- Disaster Volunteer Coordination Grid escalation views

## Backend Features
- Disaster Volunteer Coordination Grid public service APIs
- Disaster Volunteer Coordination Grid routing workflows
- Disaster Volunteer Coordination Grid SLA metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
