# Municipal Complaint Intelligence

## Summary
Municipal Complaint Intelligence is a full-stack workspace for managing municipal complaint workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Municipal Complaint Intelligence addresses a real-world gap in public operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Municipal Complaint Intelligence command dashboard
- Municipal Complaint Intelligence service board
- Municipal Complaint Intelligence escalation views

## Backend Features
- Municipal Complaint Intelligence public service APIs
- Municipal Complaint Intelligence routing workflows
- Municipal Complaint Intelligence SLA metrics

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium-Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
