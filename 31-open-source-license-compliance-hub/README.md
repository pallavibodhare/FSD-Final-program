# Open-Source License Compliance Hub

## Summary
Open-Source License Compliance Hub is a full-stack workspace for managing open source workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Open-Source License Compliance Hub addresses a real-world gap in dependency governance, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- OpenSource License Compliance Hub dependency dashboard
- OpenSource License Compliance Hub policy board
- OpenSource License Compliance Hub conflict review views

## Backend Features
- OpenSource License Compliance Hub component APIs
- OpenSource License Compliance Hub policy endpoints
- OpenSource License Compliance Hub audit summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
